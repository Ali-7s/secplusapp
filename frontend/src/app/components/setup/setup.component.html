Setup Claude API

The study platform uses Claude AI to generate questions, flashcards, explanations, and labs. You need an Anthropic API key to unlock AI features.

1

Get an Anthropic API Key

Sign up at console.anthropic.com and create an API key. New accounts get free credits.

The key starts with sk-ant-...
2

Set the Environment Variable

The backend reads your key from the ANTHROPIC_API_KEY environment variable. Set it before starting Spring Boot:

# macOS / Linux
export ANTHROPIC_API_KEY="sk-ant-your-key-here"
# Then start the backend:
cd backend && mvn spring-boot:run
# Windows PowerShell
$env:ANTHROPIC_API_KEY="sk-ant-your-key-here"
cd backend; mvn spring-boot:run
Alternatively, add it to backend/src/main/resources/application.properties:
anthropic.api.key=sk-ant-your-key-here
3

Start the Angular Frontend

cd frontend && ng serve

Frontend runs on http://localhost:4200 · Backend on http://localhost:8080
4
